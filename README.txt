== Description ==

Create beautiful schedules showing your weekly workouts. My plugin already includes the most common Workouts like Les Mills classes (Body Attack, Pump...) and also Zumba.

= Easy to use =

Tired to upload jpg scans of your schedules who doesnâ€™t fit well on small screens ?

With Fitness schedule create beautiful and customized schedules for your WordPress website and display them wherever you want with a simple shortcode Works with every themes and page builders.

= Natively responsive =

Schedules are natively responsive and will display in an elegant way which fits any screens: Desktop, tablet and smartphone.

= Customization =

Customize the colors and sizes, days of the week and start/finish hours with ease. Directly see the changes with the live preview. Developers can also customize CSS.

= Add your own workout and coachs =

Fitness Schedule comes with 13 majors workouts classes but you can add your own. Also add your coachs and assign them to workouts.

= Lightweight =

The plugin is very lightweight and only loads the minimum css/js when needed.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go in 'Fitness Planning' menu and create your first Schedule
4. A shortcode like [fitness-planning id='123'] will be created, just copy and paste it wherever you want on your website
5. You can also add new coaches and workouts via this menu

== Frequently Asked Questions ==

= Can I add my own workouts ? =

Yes! You can add as many types of workouts as you want, and customize its logo, color, and description

= Can I set 2 workouts at the same time, the same day? =

No. In order to keep it simple, it's not currently possible. But you can create several plannings and display them on the same page.

= Does it work on mobile? =

Yes! The planning is responsive and will display nicely on smartphones and tablets

= Is it compatible with every Page Builders and themes? =

Yes, I simply use shortcodes so it can be applied on every page builder like Elementor or theme like Divi.

= Can I display my plannings everywhere? =

Yes! you just have to paste the given shortcode wherever you want on the website (post, page, widgets, page builders...)

= Can I display a page with all the coaches and workouts? =

Yes! There is an option to activate the archives pages and display the list of the coaches, and another page for the workouts
