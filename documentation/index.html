Install Plugin #back to top

Manual Install

To install this plugin, unzip it and put the folder in wp-content/plugins.

Install from WP Admin

You can also install it within your admin in Plugins > Add new > Upload Plugin.

Activate plugin

Then just go to Plugins and click on Activate under the Fitness Planning name.
Now you can see the Fitness Planning entry in the left menu!

Add a Coach #back to top

The coach is the most important people in any Workouts. You can feature them in any planning you'll make, but it's absolutely not mandatory.

First go to Fitness Planning > Coachs > Add new coach

Set the name of the coach as the post title. Then import a picture of him with the WP Media library and write something inspirational about him. Don't write a too long text, let's say about 200 characters.

You'll be able to assign a coach to a Workout in Fitness Planning > Planning section.

Add a Planning #back to top

As you can see in Fitness Planning > Workouts , there are already the majors fitness classes from Les Mills and Zumba. You can edit, delete them or add your own workouts from here.

Click on Add a workout.

Set the name of the workout as the post title. Then import a logo with the WP Media Library, add an URL to the official page (optionnal) and a description.

You can also set a color for this workout. The color can be used as a background in a planning.

Create a planning #back to top

Add and Edit items

To create a new planning, go to Fitness Planning > Plannings > Add a planning. Set a name (it's just for admin purposes, it won't be displayed on site). Then start adding workouts.

Choose a Workout, a day, a start and finish hour, a coach (optionnal) and click on Add to planning

To edit an item, simply hover it and click on edit. Click on the × to delete item.

Setup planning hours and days

You can define the morning and afternoon hours, and which days should be displayed. Use the settings block below the planning preview. The preview will update in real time.

Customize look and feel

Use the settings boxes on the right side to customize the planning and workouts. You can show/hide workout logo, choose a background color or apply the custom color defined in Fitness Planning > Workouts. and you can define a border radius.

You can also define the planning background, borders and days colors. The last parameter, Pixels per Hour define how much piwels in height takes one hour. The default is 90px. You can lower this value to have a smaller planning.

If you need further customization, use the WordPress Customizer and some CSS (See FAQ).

Display a planning on your site #back to top

Displaying a planning in your site is very simple. This plugin uses WP shortcodes : Just copy the shortcode from Fitness Planning > Plannings or directly in the Add a planning view.
The shortcode looks like this [fitness-planning id='12']

Then you can paste it anywhere you want on a post, page or even in a widget. For example in a page:

You wont see directly the planning rendered, it's normal. Fitness planning works well with every page builder. But once on the site, Tada!

On small screens like smartphone and tablets, the planning will switch to the responsive view when its containers is narrower than 600px.

FAQ & Troubleshooting #back to top

How many Planning can I create? - #back to top

You can create unlimited amount of planning and display them wherever you want on your site with their shortcode.

I only see the responsive version of the planning - #back to top

It's because the area in which you try to display the planning is too narrow. You need at least 600px width to display the full planning.

I can't see the logo on some items - #back to top

The logo is hidden when the workout height is smaller than 50px. It can occur on short workouts like the 30min ones. Increase the Planning ratio to avoid this.

Further planning customizations - #back to top

If you need more customization, you can do it with CSS in the Appearance > Customize WP Panel. For example, if you want to add a border to a planning items, use .fitplan-planning-item-inside { ... }.

If you need to customize just one type of workout, check the name with your browser inspector. .fitplan-planning-item-body-balance { ... } will allow you to customize the styles only for the Body Balances classes.

Can I display a list of Workouts/coach on the site? - #back to top

Yes. Just go to Fitness Planning > Settings and enable Workouts Archive and/or Coachs Archive page.

Remember to go to Settings > Permalinks and hit save to update WP permalinks structure.

Then browse to yoursite.com/workouts/ and yoursite.com/coachs/ to see the lists.

Customizing Archive pages - #back to top

The Workouts and Coachs Archive and Single templates are located in wp-content/plugins/fitness-planning/public/templates folder. Copy theses files and paste them in your theme. If you're using a premium theme, make sure to have a child theme enabled to avoid erasing your files an on future release.

Paste these files in wp-content/themes/your-theme-child/ folder prior to edit them. You'll need the WP theming and PHP basics skills to do that.

Warning: Don't edit these files directly on the plugin folder because changes will be lost in the next update.
